# rserve-client

## Aim to support

No legacy support, only Rserve version 103

Implement RServe Commands:

    login
    assign
    eval
    OCAP
    shutdown

Implement REXP Types (not all types assignable):

    REXPDouble
    REXPInteger
    REXPGenericVector
    REXPLang
    REXPList
    REXPLogical
    REXPNull
    REXPRaw
    REXPString
    REXPSymbol
